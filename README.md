# drupal_taxonomy_hack
A simple module to correct Drupal "add term" in taxonomy.
If you have created a new vocabulary, say Book, and want to add a new term within the newly created vocabulary,  Drupal by default shows "add term" instead of "add new book". This module allows you to change the default behaviour and make it appear as expected.

Simply download and place in your modules folder.  Then enable.  You should aee that "add term" should hae changed to "add new [name of vocabulary ]
